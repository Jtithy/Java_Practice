## â–¶ï¸ How to Run the Programs

### Using Command Line

1. Install **Java JDK (8 or higher)**
2. Open terminal / command prompt
3. Navigate to the folder containing the `.java` file
4. Compile the program:

   ```bash
   javac FileName.java
   ```
5. Run the program:

   ```bash
   java FileName
   ```

### Using an IDE

You can also run the programs using:

* IntelliJ IDEA
* Eclipse
* NetBeans
* VSCode
